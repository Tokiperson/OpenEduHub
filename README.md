## 一些小规则
## Some basic rules you should follow

:white_check_mark: 你可以分享任何形式的资源，包括但不限于：docx文档、图片、学习笔记、不含敏感或版权信息的PPT、源代码、书籍等；
<br>
:white_check_mark: 如果有什么好点子或建议等，你也可以[*给我发邮件*](mailto:menghuisama@163.com?subject=建议与反馈)以告知；
<br>
:warning: 无论你准备在这里分享什么，请注意，*都**不应该**包含下面的内容*：
- 政治敏感
- 偏见或歧视
- 恐怖与反人类
- 人身攻击
- 重要的个人信息（如身份证号、你的API秘钥等）
- 无意义内容（如灌水、广告）
- 任何其他不合时宜的内容

:warning: 本仓库可能包含部分试卷或者题目文件，禁止用于学术不端行为，否则本仓库及贡献者概不负责；
<br>
:warning: 在准备提交pull request前请整理好基本的规范，如：
- Fork 本仓库
- Clone 到本地
- 创建新分支 (`git checkout -b feature/your-name`)
- 清晰简洁的命名
- 至少包含内容本体与一份记录你自己的贡献的名为`CONTRIBUTE.md`的文档，并打包为一个命名为`贡献者（可以是昵称）_主要提交内容_提交时的年月日`的文件夹
- 根据提交的内容选择仓库中合适的文件夹

:warning: 你分享的所有内容都可能被任何人学习、参考，请珍重；
<br>
:rotating_light: 本仓库的所有内容强烈不建议用于任何商业或盈利用途，仅用作个人学习、分享、交流用途；
<br>
:rotating_light: 如果您发现本仓库的任何内容侵犯了您的权益或对您构成了冒犯，请[*联系我*](mailto:menghuisama@163.com?subject=建议与反馈)。
<br>
:white_check_mark: You can share resources in any format, including but not limited to: DOCX documents, images, study notes, PowerPoint presentations that do not contain sensitive or copyrighted content, source code, books, and more；
<br>
:white_check_mark: If you have any great ideas or suggestions, feel free to [*email me*](mailto:menghuisama@163.com?subject=Suggestions%20%26%20Feedback) and let me know;
:warning: Whatever you plan to share here, please keep in mind that *it **must not** include the following*:

- Politically sensitive content  
- Bias or discrimination  
- Terror-related or inhumane material  
- Personal attacks  
- Sensitive personal information (such as ID numbers or your API keys)  
- Nonsensical content (such as spam, advertisements) 
- Any other inappropriate material

:warning: This repository may contain exam papers or question files. Use for any form of academic misconduct is strictly prohibited. The repository and its contributors assume no responsibility for such actions;
<br>
:warning: Before preparing to submit a pull request, please organize the basic specifications, such as: 
- Fork this repository
- Clone to local
- Create new branch (`git checkout -b feature/your-name`)
- Clear and concise naming 
- Include at least the main content and a document named `CONTRIBUTE.md` that records your own contributions, packaged into a folder named `Contributor (can be a nickname)_MainSubmissionContent_YYYYMMDD`
- Select the appropriate folder in the repository based on your submitted content

:warning: Everything you share here may be learned from or referenced by others—please share thoughtfully and responsibly;
<br>
:rotating_light: All content in this repository is strongly not recommended to use for any commercial or profit-making purposes. It is intended solely for personal learning, sharing, and exchange;
<br>
:rotating_light: If you find that any content in this repository infringes on your rights or causes offense, please [**contact me**](mailto:menghuisama@163.com?subject=Suggestions%20%26%20Feedback).
